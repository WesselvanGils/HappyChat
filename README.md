# HappyChat
A repository to develop a chat function for the Happy Singles project

# Setup
1. Clone this repo
2. Run npm install
3. Run npm i -g peer
4. Run the server on port 3000 with node .
5. Open a new tab in the terminal
6. Run a Peer server on port 3001 with peerjs --port 3001
